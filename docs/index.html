FAQ

If you report an issue, include repro steps, ports, and environment.

Which ports do I typically need to open?

Typical defaults are UDP 7777 (game), UDP 27015 (query), and TCP 27020 (RCON if enabled). Always confirm your configured ports.

How do I host multiple ASA instances on one machine?

Use unique ports per instance, set a unique AltSaveDirectoryName to isolate saves, and keep instance configs separated.

What does staged INI workflow mean?

Edits are written to staging, applied to the server folder on start, and a baseline is restored on safe stop, preventing half applied live INI changes.

Should I expose RCON to the internet?

No. Keep RCON LAN only or behind VPN, or restrict it via firewall allow lists to trusted admin IPs.

How do updates work?

The manager uses SteamCMD to install, update, validate the ASA dedicated server and supports operator friendly update workflows.
